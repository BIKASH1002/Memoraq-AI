# Sample Inputs for Memoraq AI

## Sample 1: Learning Framework Decision

### Decision Topic

Should I learn Django or FastAPI first?

### Situation

I know Python basics and want to build AI applications. I am confused about which backend framework will help me become job-ready and also allow me to build practical AI projects.

### Options

Option A: Learn Django first  
Option B: Learn FastAPI first

### Constraints

I can study mostly on weekends. I want to build portfolio projects within 2–3 months. I prefer practical learning over only theory.

### Confusion

Django seems better for full-stack web applications, but FastAPI seems more suitable for AI APIs and modern backend development.

## Sample 2: Laptop Upgrade Decision

### Decision Topic

Should I buy a new laptop or upgrade my current laptop?

### Situation

My current laptop is becoming slow while running Python, VS Code, browser tabs, and local AI tools. I want to build AI projects but I have limited budget.

### Options

Option A: Upgrade RAM and SSD  
Option B: Buy a new laptop

### Constraints

Budget is limited. I want a setup that can support Python development, Docker, Streamlit, FastAPI, and light local AI experimentation.

### Confusion

I am not sure whether upgrading will be enough or whether I should invest in a new laptop.
